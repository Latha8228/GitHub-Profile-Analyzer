# 🐙 GitHub Profile Analyzer

A modern web application built using **Flask** and the **GitHub REST API** that analyzes any public GitHub profile and displays repository statistics, programming languages, and profile information in a clean dashboard.

---

## 📌 Features

- 🔍 Search any GitHub username
- 👤 View profile details
- 📊 Followers, Following & Repository count
- ⭐ Total Stars
- 🍴 Total Forks
- 📈 Language Distribution Chart
- 📂 Repository List
- 🌐 Direct link to GitHub profile
- ❌ Invalid username handling

---

## ⚙ Installation

Clone the repository

git clone https://github.com/yourusername/GitHub-Profile-Analyzer.git

Move into the project

cd GitHub-Profile-Analyzer

Install dependencies

pip install -r requirements.txt

Run Flask

python app.py

Open Browser

http://127.0.0.1:5000
